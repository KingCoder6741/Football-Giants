# Real Madrid & Manchester United Fan Site

A sleek, interactive split-screen landing page dedicated to two of the biggest football clubs in the world: **Real Madrid** and **Manchester United**.

## 🚀 Live Demo
You can view the live site here: `[INSERT YOUR GITHUB PAGES URL HERE]`

## ✨ Features
- **Split-Screen Design:** Visual balance between the two legendary clubs.
- **Interactive Hover Effects:** Sections expand when hovered to focus on a specific team.
- **Dynamic Styling:** Uses iconic team colors (White/Gold for Madrid, Red/Yellow for United).
- **Responsive Layout:** Built with CSS Flexbox to work on various screen sizes.

## 🛠️ Built With
- **HTML5:** For page structure.
- **CSS3:** For styling, transitions, and layout.

## 📂 Project Structure
- `index.html`: Contains both the structure and the styles for the site.
- `README.md`: Project documentation (this file).

## 📝 How to Use
1. Clone this repository or download the `index.html` file.
2. Open `index.html` in any modern web browser.
3. To customize, simply edit the stats or links within the `index.html` file.
